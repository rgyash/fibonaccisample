# fibonaccisample 
This is a sample project to show the series of n fibonannci numbers

#Compilation
After git clone
cd to the project directory 
mvn clean package

#Run
java -jar target\fibonacci-calculator-1.0.0.0-SNAPSHOT.jar

#To test
http://localhost:8080/fibonacci/15
http://localhost:8080/fibonacci/-13
http://localhost:8080/fibonacci/0

#Current limitation
Maximum series supported is 100000
